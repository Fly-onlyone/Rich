# Themes — Map of Content

> Palettes under `@Resources/Scripts/Palettes/` provide fixed colors, while effects under
> `@Resources/Scripts/Themes/` resolve the final `BackgroundColor` / `ForegroundColor`.

## Concepts

- [[Theme System]] — how palette and effect includes are selected and what they must define
- [[Theme As Number]] — the `[EffectAsNumber]` measure used in conditionals
- [[Light Theme]] — compatibility light effect/palette values
- [[Dark Theme]] — compatibility dark effect/palette values
- [[Auto Theme]] — light/dark by system dark-mode state
- [[Color Theme]] — palette sampled from the desktop via Chameleon
- [[Blur Theme]] — palette-tinted acrylic backdrop via FrostedGlass
