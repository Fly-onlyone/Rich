# Measure

> A [[Section]] that produces a value — a number or a string — for [[Meter|meters]] and
> other measures to consume.

Measure types in Monterey include `Time`, `Calc` (arithmetic), `String` (text +
[[Substitute]]), `WebParser` (see [[WebParser]]), `Script` (Lua), and `Plugin` (see
[[Plugin]]). Measures are tagged `Group=Measures` for wholesale refresh via the
[[Group Bang Pattern]], and many drive logic through `IfCondition` actions.
