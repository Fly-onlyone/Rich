# Container

> A [[Meter]] option (`Container=OtherMeter`) that clips a meter inside the bounds and
> shape of another meter.

Rich uses containers for rounded-corner masking — e.g. the Notes widget clips its
yellow header rectangle inside a `Mask` shape so the corners stay rounded. The contained
meter is only visible where the container meter is.
