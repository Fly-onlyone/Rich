# Variables — Map of Content

> State files under `@Resources/Variables/`. These hold defaults *and* receive persisted
> user choices written by [[Settings Persistence Pattern]].

## Concepts

- [[Global Variables]] — `Global.inc`, the skin-wide defaults (theme, language, sizing…)
- [[Layout State]] — `Layout.inc`, the per-widget chosen size and active settings tab
- [[Per-Widget Variables]] — the per-widget `.inc` files (`Clock.inc`, `Weather.inc`, …)
