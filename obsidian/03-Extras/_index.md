# Extras — Map of Content

> Standalone companion skins shipped alongside the widgets. Each lives in `Extras/` and
> is its own Rainmeter config — loaded independently of the widgets.

## Companion skins

- [[AutoLowPowerMode Skin]] — toggles battery-saver mode from AC power state
- [[Sidebar Skin]] — tray/sidebar launcher for the skin's controls
- [[ThemeUpdater Skin]] — switches Light/Dark on a time schedule
- [[Unlock Skin]] — plays an unlock animation on Windows session unlock

## Supporting setup

- [[Unlock Scheduled Task]] — the Windows Task Scheduler job that triggers [[Unlock Skin]]
