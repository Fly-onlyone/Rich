# Widgets — Map of Content

> The nine end-user widgets. Each is a folder of four size `.ini` files plus shared
> logic; all of them are composed through the [[Skin Composition Flow]].

## Timekeeping

- [[01-Widgets/Clock/_index|Clock]] — analog/digital clock with selectable timezone
- [[01-Widgets/Calendar/_index|Calendar]] — month-grid calendar with Lua date logic
- [[01-Widgets/Timer/_index|Timer]] — countdown timer with alarm sound

## Media & system

- [[01-Widgets/Music/_index|Music]] — media-player control with album-art coloring
- [[01-Widgets/Monitor/_index|Monitor]] — CPU / RAM / Disk / Network / Battery gauges
- [[01-Widgets/Volume/_index|Volume]] — system master volume and per-app mixing

## Information & input

- [[01-Widgets/Weather/_index|Weather]] — Open-Meteo current + hourly + daily forecast
- [[01-Widgets/Notes/_index|Notes]] — editable sticky note
- [[01-Widgets/Reminders/_index|Reminders]] — Lua-backed to-do list with a counter
