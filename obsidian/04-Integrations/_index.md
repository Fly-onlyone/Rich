# Integrations — Map of Content

> External systems the skin depends on: one web API and the Rainmeter plugins that ship
> outside core Rainmeter.

## Web API

- [[Open-Meteo API]] — keyless weather forecast + city geocoding

## Appearance plugins

- [[FrostedGlass Plugin]] — acrylic blur backdrop for the Blur theme
- [[Chameleon Plugin]] — samples desktop/wallpaper colors for adaptive theming

## System / hardware plugins

- [[UsageMonitor Plugin]] — CPU / RAM / Disk / Network usage
- [[PowerPlugin]] — battery percentage and AC power state
- [[Win7AudioPlugin]] — system master volume control
- [[AppVolume Plugin]] — per-application volume mixing
- [[MediaPlayer Plugin]] — now-playing metadata and transport control

## Behavior plugins

- [[ActionTimer Plugin]] — stepped animation sequences
- [[InputText Plugin]] — in-skin text input fields
- [[ConfigActive Plugin]] — detects whether another skin config is loaded
- [[TrayIcon Plugin]] — system-tray icon for the sidebar
