# Contexts — Map of Content

> Right-click context-menu definitions under `@Resources/Scripts/Contexts/`. Each builds
> a menu from numbered title/action pairs; see [[Context Menu Factory Pattern]].

## Concepts

- [[Widget Context Menu]] — the default per-widget menu (edit, size, remove)
- [[Music Context Menu]] — choose the media player
- [[Monitor Context Menu]] — choose which metric to display
- [[Timezone Context Menu]] — choose the clock timezone (paginated across three files)
- [[Language Context Menu]] — choose the UI language
- [[Units Context Menu]] — choose the temperature unit
- [[Sidebar Context Menu]] — the sidebar/tray menu
- [[Tickmark Indicator]] — how the menu marks the currently-selected option
