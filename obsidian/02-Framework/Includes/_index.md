# Includes — Map of Content

> Reusable skin scaffolds under `@Resources/Scripts/Includes/`. A widget or settings
> page `@Include`s one of these to inherit chrome, theming, and the context menu.

## Concepts

- [[Widget Scaffold]] — `Widget.inc`, the base every widget builds on
- [[Window Scaffold]] — `Window.inc`, chrome for the settings window
- [[Settings Scaffold]] — `Settings.inc`, the per-widget settings-page shell
- [[Context Scaffold]] — `Context.inc`, the right-click menu wiring
- [[WhatsNew Scaffold]] — `WhatsNew.inc`, the changelog/onboarding popup
