# Meter

> A [[Section]] that draws something — text, an image, or a shape.

Meter types used in Rich include `String` (text), `Image` (icons, clock faces,
album art), and `Shape` (see [[Shape Meter]]). Meters position themselves with `X`/`Y`/
`W`/`H`, usually as expressions of widget size — the [[Proportional Layout Pattern]].
Meters are tagged `Group=Meters` so they can be refreshed together via the
[[Group Bang Pattern]].
